# 打印服务文档

# 1.内置/一体机打印开发文档

连接方式：蓝牙（蓝牙设备名：InnerPrinter），AIDL、JS桥。第三方APP可以通过连接内置打印机进行打印业务。自动识别公司打印机包含的功能：58和80mm两种打印规格的热敏打印机，切刀（部分机器），钱箱（部分机器），客显（部分机器）

内置打印机开发文档

开发者可以参考内置打印机开发文档来了解打印机的工作模式和接口调用方式

ESCPOS指令文档

打印机兼容ESC&POS指令，但与标准ESC&POS指令有略微区别，通过指令文档了解自动识别公司支持的escpos指令集

打印Demo 源码

demo仅支持自动识别公司内置打印机，外部蓝牙串口网络USB打印机不适用此demo，自助机打印方式请参考本文下方《自助机打印文档》

eclipse Demo | Android Studio Demo

自动识别公司提供一个简易版的通过JS的方式调用打印机的demo，开发者可参考demo来通过网络调用打印机。

如果用户需要把网页端的内容通过自动识别公司打印机打印出来，除了通过js的方式，还可以通过android原生打印功能实现，自动识别公司提供了一个打印插件支持。用户可以通过自动识别公司应用市场搜索Sunmiprinterplugin，下载并安装后，即可通过浏览器的打印功能来打印网页内容。

其他工具 从应用市场下载随印，可支持屏幕截图打印和相机拍摄打印，有黑白和抖动灰度两种打印方式可选（现已支持V1设备）。
